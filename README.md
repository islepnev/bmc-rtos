## BMC-RTOS

Board Management Controller

Supported boards

- CRU16 v1.0 + MCB32F769 v1.1
- TDC64VHLE v2.0
- TDC72VHL v4.2
- TTVXS v1.1

### Build

1. Download GNU Arm Embedded Toolchain and unpack
2. Set TOOLCHAIN_PREFIX variable
3. Run make
