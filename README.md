## TDC72VHL v4 BMC project

 * STM32Cube_FW_F7_V1.12.0
 * STM32F7xx HAL Driver version number V1.2.5
 * FreeRTOS 9.0

 * Tested on TDC72VHL v4.1

### Build

```
rm -rf build && make TOOLCHAIN_PREFIX=~/gcc-arm-none-eabi-8-2018-q4-major
```
